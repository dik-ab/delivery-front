================================================================================
ROUTING STRUCTURE
================================================================================

PUBLIC ROUTES:
  /login              → LoginPage
  /register           → RegisterPage

PROTECTED ROUTES (Authenticated Users):
  /dashboard          → DashboardPage (all roles)
  /trips/:id          → TripDetailPage (all roles)
  /my-matches         → MyMatchesPage (all roles)
  /tracking/:tripId   → TrackingPage (all roles)

DRIVER-ONLY ROUTES:
  /trips/new          → TripCreatePage
  /my-trips           → MyTripsPage

SHIPPER-ONLY ROUTES:
  /trips/search       → TripSearchPage

ADMIN-ONLY ROUTES:
  /admin              → AdminDashboardPage
  /admin/users        → AdminUsersPage
  /admin/trips        → AdminTripsPage

LEGACY:
  /legacy/delivery    → OldApp (backward compatibility)

DEFAULT:
  /                   → /dashboard (redirect)

================================================================================
API ENDPOINTS INTEGRATED
================================================================================

AUTHENTICATION:
  POST /auth/register         - User registration
  POST /auth/login            - User login

TRIPS:
  GET /trips                  - List all trips
  POST /trips                 - Create new trip
  GET /trips/:id              - Get trip details
  PUT /trips/:id              - Update trip
  DELETE /trips/:id           - Delete trip
  GET /trips/search           - Search with filters

MATCHES:
  GET /matches                - List matches
  POST /matches               - Create match request
  PUT /matches/:id/approve    - Approve match
  PUT /matches/:id/reject     - Reject match
  PUT /matches/:id/complete   - Complete match

TRACKING:
  POST /tracking              - Record location
  GET /tracking/:trip_id      - Get history
  GET /tracking/:trip_id/latest - Get latest

ADMIN:
  GET /admin/stats            - Platform statistics
  GET /admin/users            - All users
  GET /admin/trips            - All trips
  GET /admin/matches          - All matches
